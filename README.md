# Palirevert API

An API to get the word reversed and if it's a Palindrome or not.

## Demo

There is a working demo on: https://z5b6d8bad-z00481bbb-gtw.qovery.io

Try it with curl:

```bash
curl https://z5b6d8bad-z00481bbb-gtw.qovery.io/lol
```
